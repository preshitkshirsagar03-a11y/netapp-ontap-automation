# NetApp ONTAP Automation

## Overview

This repository contains automation scripts for managing and monitoring NetApp ONTAP storage systems using Python, PowerShell, and the ONTAP REST API.

The project demonstrates common storage administration tasks such as health monitoring, capacity reporting, volume management, snapshot management, and SVM reporting.
